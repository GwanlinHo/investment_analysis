## 創作者: Gooaye 股癌

### 影片名稱: EP625 | 🧸
#### 議題一: 台股盤勢與操作心態
開年行情火熱但操作難度高，觀察到市場主流股（如健策、貿聯-KY、智邦、台光電）出現重挫，可能暗示市場資金正在進行轉換，資金從舊主流股撤出進行新佈局。針對市場傳言（如貿聯-KY 輝達 Cableless 設計、台光電 M9 材料用量、Google OCS 棄光轉銅）進行駁斥與分析，認為多為籌碼面與資金面的調整，而非基本面惡化。

#### 議題二: NVIDIA ICMS 與存儲產業新機會
深入探討 NVIDIA 推出的 Inference Context Memory Platform (ICMS)，該技術旨在解決 Agentic AI 在推理與提取上下文時的痛點。透過 DPU 驅動的 SSD Storage Server 將 KV Cache 等準熱資料下放，節省算力與電力。在 CES 展場發現 NVIDIA Switch Tray 上採用群聯控制器（Pseudo-SLC 模式），預示存儲伺服器將迎來新機會。

#### 議題三: 記憶體題材轉向
記憶體題材焦點正從傳統 NAND/DRAM 漲價，轉向 KV Cache 下放的新應用趨勢。

### 影片名稱: EP624 | 🎥
#### 議題一: 混合實境 (MR) 隱形冠軍
分享參訪台灣未上市 MR 公司的包含，該公司技術卓越，與國際大廠及 IP 合作。預期隨著蘋果摺疊機與 AR 供應鏈發展，未來幾年 VR/AR/MR 滲透率將快速上升。

#### 議題二: 台積電 (TSMC) 展望
台積電地位穩固（GPGPU, ASIC, XPU, HBM），資本支出 (Capex) 有機會達 500 億美元以上。以 2027 年估值來看，股價仍有空間，但需留意 1 月上漲後 2 月財報展望的修正風險。

#### 議題三: AI 砍單傳言與 SRAM 概念
駁斥 Google/AWS 砍單傳言，認為是後段封測產能不足導致的出貨排程調整，而非終端需求消失。此外，華邦電與愛普因市場尋找記憶體牆解決方案（如 Pseudo SRAM/CUBE/VHM）而上漲，反映技術趨勢。
